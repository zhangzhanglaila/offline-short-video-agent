### 3. 关键功能

#### BPM自适应
```
BPM < 80    → slow (缓慢)
80-120      → medium (中等)
120-160     → fast (快速)
160+        → dynamic (极快)
```

#### 节拍对齐
- 自动找到最接近的节拍
- 计算时间偏差
- 支持自动调整事件时间

#### 同步质量评估
```
质量评分 = 对齐事件数 / 总事件数
100% = 完美同步
```

## 🎯 技术实现

### 音频分析流程
```
音频文件
  ↓
[后端检测]
  ├→ librosa (推荐)
  ├→ soundfile (备选)
  └→ ffprobe (降级)
  ↓
[特性提取]
  - 时长、采样率、声道、比特率
  ↓
[BPM估算]
  - 频谱分析 (librosa)
  - 启发式估算 (降级)
  ↓
[节拍提取]
  - 自动节拍检测
  - 节拍时间列表
  ↓
[同步输出]
```

### 同步时间线创建
```
分镜 + 音频
  ↓
分析音乐 (BPM/节拍)
  ↓
推荐节奏模板
  ↓
创建视频结构 (时间安排)
  ↓
生成同步事件
  ↓
对齐到节拍
  ↓
计算同步质量
  ↓
输出同步时间线
```
