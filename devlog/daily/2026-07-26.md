> "就是再放一张一张的ppt，没有任何交互，页面里面没有任何元素的交互，

### 根因
当前管线：每个场景渲染1张静态PNG → 定格N秒 → 拼接。
= 无帧间运动、无元素动画、字幕条压暗底部30%。

## 🎯 核心技术决策

### 渲染方案：混合方案C
- 背景运镜：FFmpeg `zoompan`（高效Ken Burns）
- 元素动画：预渲染带alpha的元素层PNG → FFmpeg `overlay`+时间表达式
- 复杂序列：局部逐帧PIL兜底
- 任何失败 → 降级现有静态渲染

### 分层模型
```
Scene → BackgroundLayer(运镜/视频)
      + OverlayLayer(轻量遮罩)
      + SubtitleLayer(入场动画)
      + ElementLayer[](逐个出现)
```
