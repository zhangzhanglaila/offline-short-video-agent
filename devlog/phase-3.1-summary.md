## 🎯 技术实现

### 转场效果流程
```
起始帧 ─┐
        ├→ 转场引擎 → 进度计算 → 效果处理 → 合成帧
目标帧 ─┘
        
效果处理:
- 淡入淡出: 像素混合
- 滑动: 区域移动
- 缩放: 图像重采样
- 模糊: 高斯模糊
- 擦除: 逐行显示
- 旋转: 旋转变换
```

### 节奏计算算法
```
总时长 = 场景时长 × 场景数 + 转场时长 × (场景数 - 1)

示例 (5个场景):
- slow:   5×5 + 1×4 = 29秒
- medium: 3×5 + 0.6×4 = 17.4秒
- fast:   1.5×5 + 0.3×4 = 8.7秒
```

### 内容分析算法
```
关键词分析 → 类型判断 → 节奏推荐

快速内容 (fast):
  音乐、舞蹈、vlog、搞笑

缓慢内容 (slow):
  冥想、美食、风景、旅游

教程内容 (medium):
  教学、讲解、指南、学习
```

### BPM自适应
```
BPM < 80     → slow
80 ≤ BPM < 120  → medium
120 ≤ BPM < 160 → fast
BPM ≥ 160    → fast
```
