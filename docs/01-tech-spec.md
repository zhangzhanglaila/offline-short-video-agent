# Offline-ShortVideo-Agent 技术设计规范

## 1. 项目定位

**一句话描述**: 全自动离线短视频生成流水线 - 选题→脚本→素材→动画→成片

**核心差异化**:
- ✅ 100%离线运行，零API成本
- ✅ 多种视觉风格，适配不同赛道
- ✅ 智能素材匹配
- ✅ 专业级动画效果

---

## 3. 核心模块设计

### 3.1 脚本生成模块 (script_module.py)
- **输入**: 主题/关键词 + 风格 + 平台
- **输出**: 结构化脚本 (hook/body/cta/storyboard)
- **LLM**: Ollama本地优先，云端降级

### 3.2 素材匹配模块 (material_matcher.py) 🆕
- **语义匹配**: 基于脚本内容匹配合适素材
- **多源支持**: Pexels/Pixabay/本地库/CC0
- **质量过滤**: 分辨率、构图、色彩评分

### 3.3 风格模板系统 (style_templates/) 🆕
```
style_templates/
├── minimal/          # 极简风格
├── vibrant/          # 活力风格
├── cinematic/        # 电影风格
├── tech/             # 科技风格
└── manga/            # 漫画风格(现有)
```

### 3.4 动画引擎 (animation_module.py)
- **转场库**: fade/zoom/wipe/scale/rotate/glitch
- **节奏控制**: per-segment timing & emphasis
- **强调动效**: big_number/chart_done/cta/hook

### 3.5 合成引擎 (composition/)
- **音频**: TTS + BGM + SFX
- **字幕**: Whisper/规则降级
- **输出**: 多平台适配

## 4. 代码规范

### 4.1 Python代码风格
- 遵循 PEP 8
- 类型注解 (Python 3.10+)
- Docstring (Google style)
- 单一职责原则

### 4.2 配置管理
```python
# 所有配置集中管理
config/
├── __init__.py       # 全局配置
├── styles.py         # 风格配置
├── platforms.py      # 平台配置
└── models.py         # 模型配置
```

### 4.3 错误处理
```python
# 统一降级策略
try:
    result = primary_method()
except Exception as e:
    logger.warning(f"Primary failed: {e}, falling back")
    result = fallback_method()
```
