# 📋 系统需求规范

## 项目概述

**项目名称**: Offline-ShortVideo-Agent (离线短视频生成系统 - Agent版)

**目标**: 构建一个基于Agent架构的智能短视频自动生成系统，支持用户输入简单需求，系统自动生成高质量的讲解类短视频。

**版本**: 2.0 (Agent架构重构版)

---

## 核心需求

### 1. 系统架构需求

#### 主控Agent（Coordinator Agent）
- **职责**: 任务拆解、子Agent分发、结果汇总、流程控制、异常处理
- **输入**: 用户需求（文本描述）
- **输出**: 完整的视频生成任务报告
- **能力**:
  - 需求理解与拆解
  - 子Agent生命周期管理
  - 任务流程编排
  - 结果校验与质量评分
  - 失败重试和异常兜底

#### 内容分析子Agent
- **职责**: 分析用户需求，生成结构化的内容大纲
- **输入**: 用户需求文本 + 视频分类 + 风格选择
- **输出**: JSON格式的内容结构
  ```json
  {
    "title": "视频标题",
    "category": "教育讲解/短视频/...",
    "style": "minimal/vibrant/...",
    "duration": 30,
    "scenes": [
      {
        "scene_id": 1,
        "title": "场景标题",
        "type": "title_card/content/conclusion",
        "text": "文字内容",
        "keywords": ["关键词1", "关键词2"],
        "duration": 5
      }
    ]
  }
  ```

#### 素材检索子Agent
- **职责**: 根据场景关键词获取素材（图片/视频）
- **输入**: 场景描述 + 关键词 + 质量要求
- **输出**: 素材列表及元数据
  ```json
  {
    "scene_id": 1,
    "materials": [
      {
        "id": "mat_001",
        "type": "image/video",
        "url": "...",
        "source": "local/pexels/pixabay/unsplash",
        "quality_score": 0.95,
        "cached": true
      }
    ]
  }
  ```

#### 视频合成子Agent
- **职责**: 根据内容和素材合成视频
- **输入**: 内容结构 + 素材 + 风格配置
- **输出**: 完整的视频文件 + 合成报告
  ```json
  {
    "video_path": "output/video_xxx.mp4",
    "duration": 30,
    "resolution": "1080x1920",
    "scenes_rendered": 6,
    "quality_score": 0.88,
    "success": true
  }
  ```

### 2. 通信协议需求

#### 消息格式（JSON）
```json
{
  "msg_id": "唯一消息ID",
  "timestamp": "2026-07-21T10:00:00Z",
  "sender": "agent_name",
  "receiver": "agent_name_or_main",
  "msg_type": "task/result/error/heartbeat",
  "task_type": "analyze/fetch_material/compose_video",
  "payload": {
    "data": {},
    "priority": 1,
    "timeout": 300
  },
  "status": "pending/processing/success/failed",
  "error": null
}
```

#### 通信方式
- 内存消息总线（MsgBus）
- 异步非阻塞通信
- 消息队列（可选Redis）

### 3. 用户交互需求

#### 输入格式
```
用户需求文本 + 分类选择 + 风格选择
例：
- 需求: "讲解Python异步编程的核心概念"
- 分类: "教育讲解"
- 风格: "minimal"
- 时长: 30秒
```

#### 输出
- 生成的视频文件
- 生成过程的日志
- 质量评分和问题提示

### 4. 视频生成需求

#### 支持的分类
- **教育讲解**: 清晰的逻辑，文字+图表，适合科普
- **短视频**: 快速切换，音效丰富，适合娱乐
- **纪录片**: 叙事性强，配乐，适合故事
- **商业宣传**: 视觉冲击，品牌展示，适合产品

#### 素材与文字展示方式
- **标题卡**: 整屏文字（通常在开头/过渡/结尾）
- **字幕条**: 下方文字 + 上方素材（讲解内容）
- **混合模式**: 根据分类自动选择

#### 素材优先级
1. 本地缓存素材（最快）
2. API素材（Pexels/Pixabay）（推荐质量）
3. 占位符/纯文本（降级方案）

### 5. 质量需求

| 指标 | 标准 | 说明 |
|------|------|------|
| **视频时长偏差** | ±2秒 | 目标时长vs实际 |
| **素材获取成功率** | ≥85% | 能找到合适素材 |
| **文字清晰度** | ≥0.8 | 文字易读性 |
| **总体质量评分** | ≥0.75 | 1-10分 |
| **生成耗时** | <120秒 | 单个视频 |

### 6. 异常处理需求

| 场景 | 处理方案 | 重试策略 |
|------|---------|---------|
| **素材获取失败** | 自动降级为纯文本卡或占位符 | 最多2次 |
| **内容分析失败** | 返回默认模板 | 最多1次 |
| **视频合成失败** | 降级为图文模式 | 最多2次 |
| **超时** | 终止该环节，跳转到兜底方案 | 无重试 |

## 非功能需求

### 性能
- 单个视频生成耗时: <2分钟
- 内存占用: <500MB
- 支持并发: ≥3个同时生成

### 可维护性
- 代码模块化，各Agent独立
- 清晰的消息协议
- 完整的日志记录

### 扩展性
- 支持新增Agent类型
- 支持新增风格模板
- 支持新增素材源

## 风险识别

| 风险 | 影响 | 缓解方案 |
|------|------|--------|
| API限流 | 素材获取延迟 | 本地缓存+多源备份 |
| FFmpeg缺失 | 合成失败 | 自动检测+降级方案 |
| 网络问题 | 整体卡顿 | 超时控制+异步处理 |
| 内存溢出 | 系统崩溃 | 流式处理+分批生成 |
